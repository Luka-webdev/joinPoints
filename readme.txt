Projekt "JoinPoints" to internetowa wersja papierowych piłkarzyków. Na samym wstępie gracz poznaje zasady gry. 

Plansza do gry narysownana jest za pomocą canvas. Zawiera kilkadziesiąt punków, po których można się poruszać. 

Użyte technologie: HTML, CSS (+SCSS), JQuery, Grunt.

Użycie Grunt pozwoliło skompresować pliki .css i .js.

W projekcie użyto google fonts.
